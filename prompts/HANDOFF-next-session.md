Ти інша модель. Провалідуй наявний design-decisions файл вимог для проєкту **Statok** і покращ де потрібно — розшир або зменш питання при потребі. Не приймай попередню роботу на віру: переглянь критично свіжим поглядом.

## Що таке Statok
Self-hosted застосунок особистих фінансів (укр. «статок» = чистий капітал / net worth). Single-user. **Фаза 1** — облік інвестицій/портфеля (акції/ETF, крипта, облігації, депозити/готівка, мультивалюта, дохідність, net worth у часі). **Фаза 2** — облік витрат + аналітика. Пріоритет — приватність (дані лише на власній інфрі).

## Твоє завдання (крок 1 пайплайну — Вимоги)
1. **Прочитай і критично оціни** `specs/statok-requirements.decisions.md`. Перевір: чи питання повні, чи нема зайвих/слабких, чи рекомендації адекватні.
2. **Розглянь пропуски, які помітила попередня модель, але НЕ додала** (додай, якщо вважаєш за потрібне):
   - Комісії/збори на угодах (fees/commissions).
   - Корпоративні дії: сплити, мерджі, зміни тікера.
   - Перекази між власними рахунками (internal transfers).
   - Стратегія бекапів/відновлення даних.
   - Метод автентифікації single-user (пароль? passkey? basic-auth за Traefik?).
   - Податкова юрисдикція/резидентність (для податкових звітів) і правила округлення.
   - Темна тема / onboarding / seed-дані для демо.
3. **Розшир або скороти** набір — ціль 20–30 якісних питань. Збережи формат design-decisions: англомовні ключові слова `Options`/`Recommendation`/`Answer`/`Comment`; `[combinable]` де можна кілька; `[assumption]` де реального вибору нема; рекомендація — 1 речення.
4. **Запусти резолв** (web UI, бо питань ≥3):
   ```bash
   TASKS_DIR="O:\projects\statok\specs" node ~/.claude/skills/design-decisions/server.mjs
   ```
   `run_in_background: true`, прочитай stdout — він друкує URL (порт авто 7777..7826), done-маркер і tasks dir. Дай користувачу URL. Чекай маркер (`until [ -f "<marker>" ]; do sleep 2; done`) або «готово» в чаті. Тоді вбий сервер.
5. **Дистиляція:** після відповідей — звести у `specs/requirements.md` (секція `## Decisions`, згрупована за темами), видалити робочий `.decisions.md`, **закомітити** (локально, без push; трейлер `Co-Authored-By: Claude <noreply@anthropic.com>`).
